# Calculator Project

Этот проект представляет собой реализацию простого калькулятора, способного выполнять математические операции: сложение, вычитание, умножение и деление. Проект написан на языке
Javascript. Он демонстрирует работу с элементами на странице, обработку событий, работу с функциями, обработку ошибок, работу с регулярными выражениями.

### Возможности:
- Сложение двух чисел;
- Вычитание двух чисел;
- Умножение двух чисел;
- Деление двух чисел;
- Обработка ошибок (деление на ноль, неверный ввод);
- Удаление старого результата по нажатию на кнопку C.

### Использование:
В приложении пользователю нужно ввести по одному числу за один раз, нажать на кнопку с операцией и нажать кнопку Равно или Enter. Ввод обрабатывается с учетом возможных 
ошибок.
